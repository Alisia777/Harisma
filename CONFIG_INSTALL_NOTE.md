# Что делать с config.js

1. Возьми файл `config.js` из этой папки.
2. Положи его рядом с `index.html` в корень репозитория.
3. Закоммить изменения и запушь на GitHub.
4. В Supabase включи `Authentication -> Configuration -> General -> Allow anonymous sign-ins`.
5. Обнови сайт и нажми:
   - `Отправить в командную базу`
   - затем в другом браузере `Загрузить командные данные`

Важно: здесь используется **publishable key**. Его можно использовать во фронтенде при включённом RLS. Не вставляй сюда `service_role` или secret key.
