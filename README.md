# Altea Portal · v7.3 Ops

Сборка под рабочий контур Алтея: pulse-дашборд сверху, задачник под РОПа, логистика по кластерам и рабочая библиотека без техдока в основном контуре.

## Что изменено в v7.3

- **дашборд пересобран по иерархии внимания**:
  - сверху сразу pulse по площадкам;
  - ниже KPI;
  - потом лидеры продаж / оборачиваемости / контента;
- **задачник работает в логике РОПа**:
  - постановка задач как в Trello / Asana;
  - приоритеты;
  - свод по сотрудникам;
  - текущая работа по артикулам;
- **логистика стала cluster-first**:
  - площадка → точка → артикулы → root cause справа;
  - кнопки цели 7 / 14 / 28 дней;
  - локальность / backlog / центральный склад / need;
- **документы очищены**:
  - сотрудникам показываются регламенты, планы, логистика, репрайсер и новинки;
  - техдок и экспорт/импорт спрятаны в админ-режим;
- **Supabase mode включён в config.js**.

## Что внутри репозитория

- `app.js` — базовый портал
- `app-imperial.js` — imperial-слой с pulse и логистикой
- `app-v71.js` — screen-архитектура и перестройка UX
- `app-v72.js` — задачник под РОПа и свод по менеджерам
- `app-v73.js` — финальная пересборка удобства, логистики и документационного слоя
- `styles.css`, `imperial.css`, `v71.css`, `v72.css`, `v73.css` — визуальные слои
- `config.js` — рабочий режим синхронизации

## Как запускать

```bash
python -m http.server 8000
```

Потом открыть `http://localhost:8000`.

## Как проверить синхронизацию

1. Открыть портал в браузере A.
2. Создать задачу в задачнике.
3. Нажать **«Синхронизировать изменения»**.
4. Открыть портал в браузере B.
5. Нажать **«Обновить командные данные»**.
6. Убедиться, что задача подтянулась.

## Админ-режим

Чтобы показать экспорт / импорт JSON и техслой, открой портал с `?admin=1`.

## Ограничения текущей версии

- график маржи по дням остаётся **операционной оценкой**, не финальной finance-grade отчётностью;
- логистический right panel строится по текущему snapshot-слою, не из live API;
- разграничение ролей по рабочим почтам ещё не закрыто на UI и RLS-hardening уровне.
