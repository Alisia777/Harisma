# Аудит версии v3

Проверено на текущем срезе данных Алтея.

## Что уже добавлено
- явный фильтр **Все площадки / WB / Ozon**;
- поиск по артикулу, названию, категории и owner;
- фильтры по owner, сегменту, внешнему трафику и закреплению;
- логика **«В работу»** по площадке:
  - WB: план < 80% + маржа WB < 0
  - Ozon: план < 80% + маржа Ozon < 0
- owner по SKU из рабочих файлов;
- иконки внешнего трафика **КЗ / VK**;
- карточка SKU с owner и каналами трафика.
