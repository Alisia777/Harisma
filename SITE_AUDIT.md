- daily margin — операционная оценка, не финзакрытие;
- для полного live-режима всё ещё нужен слой подключения рабочих файлов;
- часть пользователей может тащить старый локальный JSON из браузера — owner migration в `app-imperial.js` это чистит, но при необходимости можно вручную очистить localStorage.
