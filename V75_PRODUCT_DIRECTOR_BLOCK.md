# V7.5 · Блок «Продукт / Ксения»

Что добавлено:
- отдельный employee-first экран для Ксении как директора по товару;
- задачи в работе, статусы продукта, планы запусков и затраты;
- product agenda: ценовой коридор, план заказа заранее, позиционирование, матрица БАДов;
- pricing-risk слой из repricer для продуктового решения по коридору цены;
- Ксения добавлена в owner options для постановки задач.

Как читать экран:
1. Верх — свод по задачам, pipeline и cost coverage.
2. Слева — что в работе у директора по товару.
3. Справа — статусы продукта и постоянный focus list.
4. Ниже — план запусков и затраты.
5. Ниже — ценовой коридор и unit-экономика.

Следующий шаг:
- завести отдельный тип задач `product` / `pmr`;
- добавить product decisions и launch gates в Supabase;
- связать экран Ксении с PMR, demand review и закупкой.
