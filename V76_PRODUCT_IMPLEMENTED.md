# v7.6 · Продукт / Ксения

Что добавлено в портал:
- смена статусов товара прямо во вкладке Ксении;
- ручное заведение товаров в матрицу;
- система ввода новинок с карточкой, этапом, блокерами и next step;
- отдельный задачник Ксении со сроками;
- тетрадь действий по товарам и новинкам;
- локальное хранение + возможность синка в Supabase через отдельный SQL migration.

Чтобы новые сущности синхронизировались между пользователями, нужно один раз выполнить:
- `backend/supabase_schema_v76_product.sql`

Без migration блок Ксении всё равно работает в браузере локально и попадает в экспорт/импорт JSON.
