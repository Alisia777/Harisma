# V77.1 Fix

Что было:
- баннер писал `v76ManagerSummary is not defined`;
- сверху при этом горел `Командная база online`;
- вкладки `Продукт / Ксения` и `Ритм работы` могли уходить в пустой экран.

Root cause:
- в `app-v77.js` использовалась функция `v76ManagerSummary`, но она не была включена в сборку;
- из-за этого ломался рендер продуктового блока;
- базовый `rerenderCurrentView()` падал целиком на первом же модуле с ошибкой, поэтому следующие вкладки тоже могли не отрисоваться.

Что исправлено:
- возвращена функция `v76ManagerSummary`;
- базовый `rerenderCurrentView()` переведен на safe render с изоляцией ошибок по модулям;
- если один модуль падает, остальные экраны продолжают работать;
- баннер теперь показывает `Ошибка интерфейса`, а не маскируется под проблему данных.

Что проверить после выката:
1. hard refresh / инкогнито;
2. открыть `Продукт / Ксения` — блок должен рендериться;
3. открыть `Ритм работы` — экран должен открываться;
4. banner ошибки должен исчезнуть, если новых runtime-ошибок нет.
