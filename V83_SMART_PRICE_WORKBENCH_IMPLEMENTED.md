# V8.3 · Smart price workbench встроен в портал

Что сделано:
- вкладка `Цены` превращена из read-only matrix в рабочий `price workbench`;
- данные берутся из файла `Смарт _ Работа с ценами.xlsx` через `data/smart_price_workbench.json`;
- сохранение решений по цене работает локально и может синхронизироваться через Supabase;
- в workbench добавлены editable поля:
  - статус решения,
  - owner,
  - срок,
  - эскалация,
  - целевая цена,
  - целевая цена клиента,
  - причина,
  - комментарий,
  - флаг создания задачи;
- из решения по цене можно создавать/обновлять задачу в общем board;
- в карточке SKU появился блок `Price workbench`;
- на экране `Руководителю` появляется summary по ценовому контуру.

Что ещё важно:
- для общей синхронизации решений по цене нужно один раз прогнать `backend/supabase_schema_v83_smart_price.sql`;
- текущий `config.js`, если sync уже рабочий, лучше оставить без замены;
- historical monthly matrix остаётся как отдельный data layer; текущий модуль решает рабочий контур цены.
