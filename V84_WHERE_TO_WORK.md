# Где команда работает в единой версии портала

## 1. Работа с ценами — замена файла `Смарт _ Работа с ценами.xlsx`
Рабочее место: вкладка **`Цены / Price Workbench`**.

Здесь аналитики и владельцы решения работают по SKU:
- выбирают WB / ОЗ / ЯМ;
- открывают SKU в списке слева;
- меняют целевую цену и цену клиента;
- фиксируют причину, owner, срок и комментарий;
- ставят эскалацию (`Ксении`, `Финальное согласование`, `РОПу`, `Финансам`);
- при необходимости создают задачу.

Это и есть основная замена Excel по ценам. Excel остаётся source / backup.

## 2. Машина форма — замена отдельного файла статусов
Рабочее место: вкладка **`Задачи и контроль`**, блок **`Форма статусов WB · РОП (замена файла Маши)`**,
и карточка конкретного SKU — блок **`Форма статусов РОПа WB`**.

Здесь РОП:
- создаёт новую статусную запись;
- фиксирует риск, next step, комментарий, срок;
- при необходимости превращает статус в задачу.

## 3. Чек-лист запуска новинок — явный блок для Маши
Рабочее место: вкладка **`Продукт / Ксения`**, блок **`Запуски / Чек-листы новинок`**.

Здесь видно:
- по каким новинкам чек-лист уже создан;
- сколько задач done / всего;
- просрочку;
- конкретные launch-задачи;
- можно создать или обновить чек-лист по нужной новинке.

## 4. Важное правило
Портал — рабочая система, не витрина. Данные могут приходить из файлов и ETL, но:
- решения по цене фиксируются в `Цены`;
- WB-статусы РОПа фиксируются в форме Маши внутри портала;
- чек-листы запуска ведутся в явном блоке `Запуски / Чек-листы новинок`.
