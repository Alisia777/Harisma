# Что грузить для обновления до v8.5

Грузи **только один пакет**: `brand_portal_altea_v85_launch_control_unified.zip`.

## Перед загрузкой
1. Сделать backup текущего репозитория.
2. Сохранить текущий рабочий `config.js`.
3. Не удалять `CNAME`.

## Обновление репозитория
1. Распаковать архив.
2. Залить содержимое в корень репозитория.
3. Если sync уже рабочий — оставить свой текущий `config.js`.
4. Commit → Push.

## Supabase
Если уже выполнялся `v84_unified.sql`, можно выполнить только:
- `backend/supabase_schema_v85_launch_control.sql`

Если ставится всё с нуля — выполнить:
- `backend/supabase_schema_v85_unified.sql`

## После выката проверить
1. Появилась вкладка `Запуск новинок`.
2. Вкладка `Продукт / Ксения` стала легче и не содержит большой launch-board.
3. В `Запуск новинок` видно реестр запусков, фазы и чек-лист.
4. Сохраняются поля owner / риск / next step.
5. Создаётся checklist и его задачи видны.
