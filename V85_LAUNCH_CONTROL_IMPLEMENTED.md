# V8.5 · Запуск новинок вынесен отдельно

Что сделано:
- добавлена отдельная явная вкладка `Запуск новинок` для Маши;
- launch-checklist вынесен из тяжёлого блока `Продукт / Ксения`;
- в новой вкладке есть:
  - реестр запусков,
  - summary по рискам и просрочкам,
  - фильтры,
  - подробная карточка запуска,
  - фазы чек-листа,
  - inline-обновление статуса задач запуска;
- сохранение метаданных запуска идёт локально и может синкаться в Supabase через таблицу `portal_novelty_launches`;
- блок Ксении разгружен: вместо launch-board в нём теперь только ссылка-переход на отдельную вкладку запуска.

Что проверить после выката:
1. Появилась новая вкладка `Запуск новинок`.
2. В `Продукт / Ксения` больше нет тяжёлого блока checklist, есть явная ссылка в новый раздел.
3. В `Запуск новинок` виден реестр запусков.
4. Для выбранной новинки можно:
   - сохранить owner/статус/риск/next step,
   - создать или обновить checklist,
   - менять статусы launch-задач.
5. После refresh запуск и его поля не пропадают.
